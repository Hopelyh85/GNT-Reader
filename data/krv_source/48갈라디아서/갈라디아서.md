---

- **문체 및 어조:** 활기차고, 직설적이며, 공격적이고, 직접적임. 교정적, 긴박함, 간결함, 의로운 분노, 강력한 어조.
    
- **주제:** 율법의 행위가 아닌 믿음으로 말미암는 의 (이신칭의).
    
- **저자:** 바울 (큰 글자로 직접 씀, 갈 6:11).
    
- **수신자:** 갈라디아에 있는 교회들 (갈 1:2) (아마도 남부 지역).
    
- **갈라디아서에 나타난 그리스도:** 예수님은 믿는 자의 새로운 생명의 근원이자 능력이심 (갈 2:20, 5:16).

### 은혜와 율법의 대조

| **율법 (LAW)**                | **은혜 (GRACE)**                    |
| --------------------------- | --------------------------------- |
| 율법은 금지한다.                   | 은혜는 초대하고 베푼다.                     |
| 율법은 죄인을 정죄한다.               | 은혜는 죄인을 구속한다.                     |
| 율법은 "행하라"고 말한다.             | 은혜는 "다 이루어졌다"고 말한다.               |
| 율법은 "계속 거룩하라"고 말한다.         | 은혜는 "다 이루었다"고 말한다.                |
| 율법은 저주한다.                   | 은혜는 복을 준다.                        |
| 율법은 죄인을 죽인다.                | 은혜는 죄인을 살린다.                      |
| 율법은 하나님 앞에서 모든 입을 막는다.      | 은혜는 하나님을 찬양하도록 입을 연다.             |
| 율법은 가장 선한 사람도 정죄한다.         | 은혜는 가장 악한 사람도 구원한다.               |
| 율법은 "네 빚을 갚으라"고 말한다.        | 은혜는 "내가 너의 모든 것을 값없이 용서하노라"고 말한다. |
| 율법은 "죄의 삯은 사망"이라고 말한다.      | 은혜는 "하나님의 은사는 영생"이라고 말한다.         |
| 율법은 "범죄하는 그 영혼은 죽으리라"고 말한다. | 은혜는 "믿고 살라"고 말한다.                 |
| 율법은 죄를 드러낸다.                | 은혜는 죄를 속한다.                       |
| 율법으로는 죄를 깨닫게 된다.            | 은혜로는 죄에서 구속함을 받는다.                |
| 율법은 모세로 말미암아 주어졌다.          | 은혜와 진리는 예수 그리스도로 말미암아 왔다.         |
| 율법은 순종을 요구한다.               | 은혜는 순종할 수 있는 능력을 부여하고 베푼다.        |
| 율법은 돌판에 기록되었다.              | 은혜는 마음 판에 기록된다.                   |
| 율법은 그리스도 안에서 폐하여졌다.         | 은혜는 영원히 거한다.                      |
| 율법은 우리를 종노릇 하게 한다.          | 은혜는 우리를 하나님의 아들들의 자유 안에 둔다.       |
